Para iniciar un proyecto en node es asi:
node init 

Para instalar un paquete y ponerle --save quiere decir que sera necesario como una 
dependencia

npm i yargs --save 
este paquete sirve para mejor la parte de ingresar parametros desde la 
aplicacion

Para instalar un paquete que se solo local para la aplicacion es de 
esta forma para que no sea global

npm install --save-dev nodemon

para desintalar un paquete es de la siguiente forma

npm uninstall nodemon
